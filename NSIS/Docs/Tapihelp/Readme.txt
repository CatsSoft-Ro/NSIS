TAPI Helper Plugin

tapihelp.dll - medium (22k), simple plugin that lets you explore TAPI
subsystem and report installed modem name and modem device id.

Functions:

tapihelp::getmodemid
	returns modem device id on stack (to get it use Pop $var)

tapihelp::getmodemname
	returns modem name on stack (to get it use Pop $var)
